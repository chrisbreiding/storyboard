## Storyboard

A read-only client for Pivotal Tracker optimized for use on a status board.

### Contributing

Install:

* node
* npm

Get dependencies:

```
npm install -g gulp
npm install
```

Start developing:

```
npm run dev
```

which will:

* compile CoffeeScript, SCSS, and Handlebars(.hbs) files on save
* run a server at http://localhost:8080
